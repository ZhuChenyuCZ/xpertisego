# Xpertise-backend

## 运行

- Django backend
  - Run in `localhost:8000`

```bash
pip install django # default latest version

# django-backend 单页django应用
python search.py runserver

# pybackend # 完整django项目
python manage.py runserver
```

- Flask backend
  - Run in `localhost:8080`

```bash
pip install flask
python main.py
```

- Gin backend
  - Run in `localhost:8080`

```bash
go mod tidy
go run main.go
```
